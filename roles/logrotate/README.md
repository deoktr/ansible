# logrotate

Configure logrotate.

## Variables

| name                  | type | default  |
| --------------------- | ---- | -------- |
| `logrotate_frequency` | str  | `weekly` |
| `logrotate_keep`      | int  | `4`      |
| `logrotate_dateext`   | bool | `true`   |
| `logrotate_compress`  | bool | `false`  |
