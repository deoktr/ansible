# SSH

Configure OpenSSH.

-   Install OpenSSH
-   Configure OpenSSH
-   Manager SSH permissions with group

## Variables

| name              | type      | default     | description                                             |
| ----------------- | --------- | ----------- | ------------------------------------------------------- |
| `sshd_port`       | int       | `22`        |                                                         |
| `sshd_users`      | list[str] | `[]`        |                                                         |
| `sshd_group_name` | str       | `sshusers`  |                                                         |
| `sshd_from_ip`    | list[str] | _local_net_ | list of IP or IP ranges that will be able to access SSH |
